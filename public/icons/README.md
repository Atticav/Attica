# Ícones PWA da Attica

Coloque os ícones da marca nesta pasta com os seguintes nomes e tamanhos:

- icon-192x192.png  → 192×192px (Android)
- icon-512x512.png  → 512×512px (Android splash)
- icon-180x180.png  → 180×180px (Apple Touch Icon / iPhone)

Use o logo circular da Attica (fundo bege claro #FAF6F3, emblema marrom).
Os arquivos devem estar em formato PNG.
